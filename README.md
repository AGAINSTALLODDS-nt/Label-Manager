Приложение доступно по адресу: `[https://againstallodds-nt.github.io/label-manager/](https://againstallodds-nt.github.io/Label-Manager/)`

## Использование

### 1. Регистрация/Вход
- Создайте аккаунт или войдите в существующий
- Данные хранятся локально в браузере (localStorage)

### 2. Создание этикеток
- Нажмите "➕ Добавить этикетку"
- Заполните обязательные поля (Артикул, Штрихкод)
- Можно сгенерировать случайный штрихкод
- Укажите количество для печати

### 3. Импорт из Excel
- Нажмите "📥 Загрузить из файла"
- Перетащите файл или выберите через кнопку
- Проверьте предпросмотр данных
- Нажмите "Импортировать"

**Формат Excel файла:**
| Артикул | Штрихкод | Цвет | Размер | Название товара | ... | Количество |
|---------|----------|------|--------|-----------------|-----|------------|
| ART001  | 4601234567890 | белый | XL | Футболка | ... | 10 |

### 4. Управление этикетками
- **Редактирование**: нажмите ✏️
- **Дублирование**: нажмите 📑
- **Копирование штрихкода**: нажмите 📋
- **Удаление**: нажмите 🗑️
- **Изменение количества**: используйте +/− или введите вручную

### 5. Массовые операции
- Отметьте нужные этикетки чекбоксами
- Используйте панель массовых действий внизу экрана

### 6. Печать
- Выберите этикетки для печати
- Нажмите "🖨️ Печать выбранных"
- Настройте параметры печати:
  - Формат штрихкода
  - Размер этикетки
  - Размер текста
  - Ориентацию
- Проверьте предпросмотр
- Нажмите "🖨️ Печать"

## Технологии

- **HTML5/CSS3** - структура и стили
- **Vanilla JavaScript** - логика приложения
- **JsBarcode** - генерация штрихкодов
- **SheetJS (XLSX)** - работа с Excel файлами
- **localStorage** - хранение данных

## Безопасность

- Все данные хранятся локально в браузере пользователя
- Пароли хранятся в открытом виде (для локального использования)
- Для продакшена рекомендуется использовать серверную авторизацию
